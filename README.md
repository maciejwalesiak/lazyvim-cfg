Config template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

Run :LazyHealth, install missing tools (but do not install eg. fdclone, it breaks down 'find' functionality).

Steps to enable rust:
1. download latest [rust-analyzer](https://github.com/rust-lang/rust-analyzer/releases), install it to ~/.cargo/bin/rust-analyzer
2. enable rust lang in lazyVim's LazyExtras
3. restart nvim
